# whitepaper-simplifier

An AI powered tool that converts complex whitepapers into simple summaries

Features:
- Plain English summary
- Key concepts breakdown
- Tokenomics insights
- Risk & red flag detection
- Glossary of important terms

Powered by OpenGradient LLM SDK
